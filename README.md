# osmosis-proto

Protobuf type definitions and clients of the Osmosis chain, in Rust.

## How to use

Edit the `OSMOSIS_REV` constant in `./proto-build/src/main.rs` to the desired Osmosis version, then:

```bash
cd proto-build
cargo +nightly run
```
